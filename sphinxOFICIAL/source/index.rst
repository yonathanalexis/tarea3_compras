Bienvenido a la Documentación 
=========================================
En la siguiente documentación presentaremos el desarrollo integro
y general sobre la aplicación
Esta aplicacion es creada y destinada a la administracion
de compras de una empresa, permite que una empresa mantenga un registro de su stock de productos
ademas de formularios de adquisición de productos tomando en cuenta el proveedor y los descuentos 
que este puede entregar

Introducción
-----------------------------------------
Nuestro programa desarrollado en python utiliza pyside y qtdesigner para
la interfaz y entorno grafico de la aplicación. además mediante la implementación 
de base de datos permite un manejo dinámico de la información.
